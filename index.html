What LUMI OS provides

Core OS

「今より悪くならない」ための床 OS

毎月の固定費・最低生活費・安全バッファから、 あなた専用の床 paket_bigzoon を計算。 アプリを使うほど、「床を割りにくい体質」に近づくことだけを約束します。

Aurora loupe

手数料を“増やす”のではなく、“見える化”する

LUMI は銀行や Wise と競争しません。
既存サービスの上に、「どれだけ損しているかを映すルーペ」として乗るだけです。

  • レバレッジ・約束利回り・借金スパイラルは提供しない
  • 紹介料が発生しても、ユーザーの手数料を上乗せしない
Behavior brake

心のコンディションも OS に載せる

疲れている日・感情が揺れている日は、お金の判断もブレやすい。 LUMI OS は、簡単な質問から NORMAL / TIRED / RED を判定し、 RED なら大きな決断を一旦止めるように設計します。

For whom

・フリーランス / 個人事業主

売上が月ごとに変動し、「いつもなんとなく不安」。 床を固定し、そこからの距離だけを見ることで、 「今月どこまで動かしていいか」が一目でわかります。

・海外送金 / 多通貨を使う人

為替レート・中継銀行・サービス手数料…。 Aurora ルーペは、それらをまとめて 「実質いくら損しているか」で見せます。

・「投資よりまず生活の安定」が大事な人

LUMI OS は、レバレッジや高リスク商品の販売を行いません。 まずは「床を割らない」ことだけに集中したい方のための OS です。
