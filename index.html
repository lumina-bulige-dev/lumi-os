About LUMI OS

LUMI OS は「床（financial floor）」と「見えない損（hidden cost）」を扱う 世界初のスタビライザー型 OS です。 使うほど「今より悪くならない」構造だけを提供します。

MVP Features

  • ✓ paket_bigzoon（床）の可視化
  • ✓ SAFE_NULL：無行動で合格できる日
  • ✓ heart_mode：NORMAL / TIRED / RED
  • ✓ Aurora：手数料ルーペ（見えない損を可視化）
  • ✓ state → reaction（LUMI_ALGO 心臓部）
